今天煮什麼（CookToday）

AI 驅動的智慧食譜推薦 App

關於此 App

「今天煮什麼」是一款 AI 驅動的智慧食譜推薦 App。拍下冰箱裡的食材，AI 即時為你量身打造食譜，支援中、西、日、韓、泰式料理及多種特殊飲食需求。

常見問題（FAQ）

需要自備 API Key 嗎？

是的。請在 App 內「設定」頁面輸入您的 OpenAI 或 Groq API Key。API 金鑰以 iOS Keychain 加密儲存，安全無虞。

支援哪些語言？

App 介面支援繁體中文、English、日本語、한국어，可在設定中即時切換，無需重新啟動。

支援哪些料理風格？

目前支援中式、西式、日式、韓式、泰式共 5 種料理風格。

有特殊飲食需求怎麼辦？

App 提供 9 種特殊飲食模式，包含兒童餐、銀髮餐、低脂健身、高蛋白增肌、低醣、術後恢復、糖尿病友善、低鈉及低渣飲食。請注意，AI 生成的食譜僅供參考，如有醫療相關飲食需求，請諮詢專業醫師或營養師。

食譜可以儲存嗎？

可以。每次生成的食譜會自動儲存至「食譜紀錄」，您也可以為成品拍照、一鍵複製分享。

為什麼食譜生成失敗？

請確認以下事項：

  1. 網路連線是否正常
  2. API Key 是否正確輸入
  3. API 帳戶是否仍有額度

您可以在「設定」頁面點擊「連線測試」確認 API 狀態。

我的資料安全嗎？

您的 API 金鑰以 iOS Keychain 安全儲存，所有 API 通訊皆透過 HTTPS 加密傳輸。食材與食譜資料僅儲存在您的裝置上，不會上傳至任何伺服器。
